# Icons Needed

Please create three PNG icon files for the extension:

## Required Files

1. **icon16.png** - 16x16 pixels (toolbar icon)
2. **icon48.png** - 48x48 pixels (extension management page)
3. **icon128.png** - 128x128 pixels (Chrome Web Store)

## Design Suggestions

- Use a speech bubble or comment icon
- Vibrant colors (purple, pink, blue gradients)
- Match the gamified theme of the widget
- Simple and recognizable design

## Quick Creation Options

### Option 1: Online Tools
- [Canva](https://www.canva.com) - Free icon creator
- [Favicon.io](https://favicon.io) - Convert emoji to icon
- [IconScout](https://iconscout.com) - Free icon generator
